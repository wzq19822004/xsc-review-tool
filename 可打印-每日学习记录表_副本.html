📐 数学核心公式速记卡
模块公式/方法
等差数列aₙ=a₁+(n-1)d  |  Sₙ=n(a₁+aₙ)/2  |  n=(aₙ-a₁)/d+1
裂项法1/[n(n+1)] = 1/n - 1/(n+1)
分数应用题对应量÷分率=单位"1"  |  单位"1"×分率=对应量
经济问题售价=成本×(1+利润率)  |  利润=售价-成本
浓度问题浓度=溶质÷溶液×100%  |  混合前后溶质不变
工程问题工作效率×时间=总量  |  合作时间=1÷(1/t甲+1/t乙)
牛吃草每天新长草=(多吃天数×牛数-少吃天数×牛数)÷天数差
相遇问题S=(v₁+v₂)×t
追及问题ΔS=(v₁-v₂)×t
流水行船V顺=V静+V水  |  V逆=V静-V水
圆面积/周长C=2πr  |  S=πr²
圆柱/圆锥V柱=πr²h  |  V锥=πr²h/3
鸡兔同笼兔数=(总腿-2×总头)÷2
🔤 英语语法核心速记
语法点核心结构/口诀
6种时态一般现在(do/does) / 一般过去(did) / 将来(will do) / 现在进行(be doing) / 过去进行(was/were doing) / 现在完成(have/has done)
被动语态be + done（时态看be：am/is/are+done / was/were+done / will be+done / have been+done）
宾语从句连接词 + 陈述语序 + 时态一致（主过从必过）
定语从句who(人主) / whom(人宾) / which(物) / that(通用) / whose(…的)
状语从句主将从现！if/when从句用一般现在时表将来
比较级A+比较级+than+B / the+比较级, the+比较级（越…越…）
情态动词can/could→能力; may/might→可能; must→必须/一定(can't否定); should→应该
不定代词some-(肯定/请求) / any-(否定/疑问) / no-(否定含义) / every-(每个)
