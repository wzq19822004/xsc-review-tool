# 小升初分班考 · 12天系统复习工具

王瑾（六年级）专用，7月分班考复习工具。

## 🚀 使用方法

用浏览器打开 `index.html` 即可使用。无需安装、无需服务器。

分享给别人：
- **单文件版**：`dist/小升初复习工具.html`（504KB，一个文件直接打开）
- **打包版**：`dist/小升初复习工具.zip`（125KB，解压后打开）
- **多文件版**：`dist/小升初复习工具-多文件版.zip`（131KB，完整项目）

构建命令：`python3 build.py`（生成 dist/ 目录下的单文件版）

## 📊 功能特性

| 功能 | 说明 |
|------|------|
| 📊 Dashboard | 三科进度可视化、每日分数追踪、下一步建议 |
| 📖 每日学习 | 知识点展开/折叠、分层练习（点选项即判题） |
| 📋 考察测验 | 自动判分、每题显示完整原题+所有选项+解析 |
| 📝 错题本 | 错题自动收录、同类题重测（答对即消除） |
| 📊 学习日报 | 每日知识点清单+测验详情+复习建议，可打印 |
| ✅ 打卡 | 每日任务勾选，localStorage 持久化 |
| 🖨️ 打印 | @media print 优化，一键打印当天内容 |
| 🔄 重新测验 | 清除当天成绩重新做 |

## 📚 课程内容

### 语文（12天，51个知识点，251道练习，96道测验）
- 基础运用：字音、字形、词语运用、关联词、病句、标点、对联、语言得体、句式变换、传统文化
- 诗词鉴赏：12首经典诗词（观沧海→泊秦淮）全文+注解+背景+手法+名句赏析
- 文言文：12篇（论语→赵高指鹿为马）原文+翻译+重点字词+寓意分析
- 阅读理解：10种题型，每篇带完整原创阅读篇章
- 名著导读：四大名著（朝花夕拾、西游记、水浒传、红楼梦）
- 作文：小作文/续写/命题/半命题

### 数学（12天，43个知识点，168道练习，120道测验）
- 六大模块：计算(~20%)、数论(~9%)、分百应用题(~21%)、工程问题、比例与方程(~36%)、行程问题、几何、排列组合
- 覆盖86%以上考点，含公式推导和典型例题

### 英语（12天，34个知识点，173道练习，120道测验）
- 语法+阅读+写作+词汇四位一体
- 完整中考语法框架：时态、被动语态、宾语/定语/状语从句

**总计：928道练习题+336道测验题**

## 🔧 技术方案

- vanilla HTML/CSS/JS，零依赖
- 深色主题，16px基础字号，1.85倍行距
- localStorage 持久化（打卡/成绩/错题本）
- 响应式布局（桌面+平板）
- 打印友好（@media print）
- 多文件开发，浏览器直接打开即用

## 🎯 给王瑾的使用建议

1. 每天先学知识点（展开阅读），再做分层练习
2. 做完练习后做考察测验（错题自动收录）
3. 每天打卡4项任务
4. 定期查看错题本，做同类题重测
5. 周末打印学习日报复习
