### 方式一：浏览器直接打开（推荐）

或在 Finder 中双击 `小升初复习工具.html`。

将文件通过 AirDrop / 微信发送到设备，用 Safari 打开即可。

## 💡 使用建议

### 每天学习流程
1. 打开工具 → 选择科目 → 选择当天Day
2. **展开知识点卡片** → 阅读精炼总结
3. **看视频** → 点击B站/Douyin链接学习
4. **做练习** → 完成每日作业
5. **打卡** → 在checklist中逐项勾选，最后点"打卡"按钮

### 学习节奏建议
- 距离考试30天以上：每天1个Day（语数英各1天=3天量/天，12天学完一轮）
- 距离考试15-30天：每天2个Day（语数英各2天，6天学完一轮）
- 距离考试<15天：每天3个Day + 真题模拟，重点看薄弱环节

### 打印使用
- 点击"🖨️ 打印当天"可以打印当天所有学习材料
- 打印版自动隐藏导航栏，适合纸质学习
- 建议打印出来做练习和背诵

## ⚠️ 注意事项

- **数据存储**：打卡记录保存在浏览器localStorage中，清除浏览器数据会丢失打卡记录
- **视频链接**：B站/Douyin链接是搜索链接，点击后需要手动选择合适视频
- **离线使用**：本工具完全离线可用（视频链接需要联网）
- **兼容性**：推荐Chrome/Safari/Edge浏览器
