# 小升初分班考复习工具 — 实施计划

> **For Hermes:** 连续执行本计划，直到产品开发完成并验证通过。

**目标：** 为王瑾（六年级，7月分班考）打造一套涵盖语文/数学/英语三科、每科12天的在线复习与打卡工具。

**架构：** 单文件HTML应用（内联CSS/JS），localStorage做打卡持久化，数据驱动渲染，打印友好CSS。

**技术栈：** HTML5 + CSS3 + Vanilla JS，零依赖，浏览器直接打开即用。

---

## 产品定位

### 用户画像
- **主要用户：** 王瑾（12岁，六年级男生，视觉学习型）
- **辅助用户：** Gavin（父亲，监督进度、打印材料）
- **使用设备：** 电脑/iPad 浏览器，偶尔打印

### 核心需求
1. **每日学习导航** — 按天查看当天要学什么（知识点、诗词、文言文等）
2. **知识点精讲** — 每个知识点的核心总结+典型例题
3. **视频辅助** — B站/抖音对应知识点讲解视频链接
4. **打卡追踪** — 完成每日任务后打卡，记录进度
5. **打印友好** — 可打印每日学习材料和练习题
6. **线上使用** — 浏览器直接打开，无需服务器

### 内容范围
- **语文 12天：** 基础运用/诗词鉴赏/文言文/阅读理解/名著导读/作文
- **数学 12天：** 计算/数论/分百应用/工程/比例方程/行程/几何/排列组合
- **英语 12天：** 语法体系/阅读题型/写作训练/词汇积累

## 开发任务

### Task 1: 项目骨架 — HTML结构 + CSS基础样式
**目标：** 创建单文件HTML，搭建三科Tab导航 + 侧边栏 + 主内容区布局

### Task 2: JavaScript数据模型 — 课程内容数据结构
**目标：** 将36天课程内容（语文12天+数学12天+英语12天）填充到COURSES数据结构

### Task 3: 每日内容渲染引擎
**目标：** 基于COURSES数据动态渲染每日课程内容（知识点卡片+展开/折叠）

### Task 4: 打卡系统
**目标：** localStorage实现每日打卡，进度条实时更新

### Task 5: 视频链接区 + B站搜索
**目标：** 为每个知识点搜索对应的B站/Douyin视频链接

### Task 6: 打印样式 + 可打印练习册
**目标：** @media print优化，可一键打印当天内容

### Task 7: 进度仪表盘
**目标：** 总览页面显示三科完成进度

### Task 8: 测试验证
**目标：** 浏览器打开验证所有功能，修复bug
